# ColorScheme

The palette of predefined colors for a page.

## Schema

```json
{
  "colors": array of [ThemeColorPair]
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `colors` | array of [ThemeColorPair] | The ThemeColorType and corresponding concrete color pairs. |
