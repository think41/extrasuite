# TextRun

A TextElement kind that represents a run of text that all has the same styling.

## Schema

```json
{
  "content": string,
  "style": [TextStyle]
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `content` | string | The text of this run. |
| `style` | [TextStyle] | The styling applied to this run. |
