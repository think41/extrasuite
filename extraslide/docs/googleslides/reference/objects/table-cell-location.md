# TableCellLocation

A location of a single table cell within a table.

## Schema

```json
{
  "rowIndex": integer,
  "columnIndex": integer
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `rowIndex` | integer | The 0-based row index. |
| `columnIndex` | integer | The 0-based column index. |
