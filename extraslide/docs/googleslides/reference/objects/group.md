# Group

A PageElement kind representing a joined collection of PageElements.

## Schema

```json
{
  "children": array of [PageElement]
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `children` | array of [PageElement] | The collection of elements in the group. The minimum size of a group is 2. |
