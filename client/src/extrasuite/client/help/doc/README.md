Google Docs - edit documents using GitHub-flavored markdown.

pull downloads one markdown file per tab. Edit them, then push to sync back.

  extrasuite doc pull <url> [output_dir]    Download document as markdown
  extrasuite doc push <folder>              Push local changes to Google Docs
  extrasuite doc create <title>             Create a new document
  extrasuite doc share <url> <emails>       Share with contacts

Always re-pull after pushing before making more changes.

For XML format: use pull-xml / push-xml instead.
